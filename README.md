# IMPS-DRT-fitting

This repository is intended to host workflows related to IMPS fitting and DRT-based analysis.

## Current status

At present, this repository includes a reference notebook, `Lasso DRT.ipynb`, which is retained for methodological comparison and learning purposes.

This notebook is associated with the upstream Lasso-based IMPS-DRT workflow developed in the following repository:

- [nrex-unibo/LassoDRT](https://github.com/nrex-unibo/LassoDRT)

## Repository development

A dedicated MATLAB workflow will be added to this repository in a separate folder to provide:

- MATLAB scripts for IMPS data preprocessing
- regularized fitting routines
- demo datasets
- visualization examples
- step-by-step instructions for use

## Purpose

The purpose of this repository is twofold:

1. to preserve a reference implementation related to Lasso-based IMPS-DRT analysis  
2. to build a reusable MATLAB workflow for IMPS fitting and analysis

## Attribution

The reference notebook currently included in this repository is related to the upstream project:

- [nrex-unibo/LassoDRT](https://github.com/nrex-unibo/LassoDRT)

Please refer to the original repository and the associated publication for the original implementation details and methodological background.

## Note

The reference notebook and the MATLAB workflow serve different roles in this repository:

- `Lasso DRT.ipynb` is kept as upstream reference material
- the forthcoming `matlab/` folder will contain the MATLAB-based workflow developed in this repository
